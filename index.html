What is Claude?

Claude is an AI assistant made by a company called Anthropic. You type a question or a request, and it writes back in plain language, the same way a knowledgeable colleague would.

It can answer questions, write and edit documents, explain things clearly, plan trips, summarise long files, and a lot more. You do not need to know anything technical to use it. If you can send a text message, you can use Claude.

Three Ways In

Where to Find Claude

The Claude app for iPhone and Android, including voice.

Your First Minute

Open It and Type

Sign up at claude.ai with your email. You land on a screen with one big text box. Type a real question, press enter, and read the reply.

There is no wrong first message. Ask it to explain something, draft an email, or plan your week. To start over on a new topic, open a new chat so the old conversation does not get in the way.

The Model Picker

Which Claude Should I Pick?

Near the top of the screen you can choose a model. Each one trades speed for depth. When in doubt, leave it on the default and carry on.

Haiku

Fast and light

Quick answers and simple questions. The speedy option.

Sonnet

The daily driver

A balanced all-rounder for writing, planning, and everyday work.

Opus

Deep thinker

Heavier reasoning for harder, more detailed tasks. On paid plans.

Fable

Most capable

For the biggest, most involved projects. On paid plans.

Beginner rule of thumb: Sonnet for most things, faster Haiku when you are in a hurry.

Stay Organised

Projects:
A Folder for One Topic

A Project is a workspace that keeps everything about one topic in one place: its chats, its files, and a set of instructions Claude follows every time.

Say you run a small bakery. Make a "Bakery" project, drop in your menu and price list, and tell Claude to always answer in a warm, friendly tone. From then on, every chat inside that project already knows the background.

Free accounts can make a handful of projects. It is the simplest way to stop re-explaining yourself.

A Quick Tour

What's on the Screen

It looks busier than it is. There are only five things to know.

1
The sidebar. Every past chat and project, down the left. Click one to reopen it.
2
New chat. Starts a fresh, blank conversation for a new topic.
3

4

5
Settings. Where you turn features on and find the keyboard shortcuts.

Work Faster

A Few Keys Worth Knowing

You never have to use these. But once they stick, the app feels twice as quick. On a Mac use the Cmd key; on Windows use Ctrl.

Cmd+N

Start a new chat

A blank conversation, instantly, without reaching for the mouse.

Enterto sendShift+Enter

Send, or add a new line

Enter sends your message. Hold Shift and press Enter to write a second line first.

Cmd+K

Jump to any chat

Opens a search box. Type a few words and hop straight to an old conversation.

⌥+Space

Summon Claude anywhere

Set a hotkey that pops Claude open over whatever you are doing, then hides it again.

Not sure of a key on your machine? Open Settings and the app lists every shortcut, including how to set the summon hotkey.

Hands-Off Work

Let Claude Work
in the Background

The desktop app has a roomier space called Cowork. It is for bigger jobs, the kind that take a few steps, so Claude can keep working while you get on with something else.

This is also where scheduled tasks live: jobs you set up once and Claude runs on a timer, like a news brief every morning or a summary every Monday. We will set one up together in a few slides.

Think of the normal chat box as a quick conversation, and Cowork as handing over a small project and walking away.

Good Habits

How Regulars Actually Use It

None of this is complicated. These are the small habits that turn the app from a novelty into something you lean on daily.

Leave it open

Keep the app running in the background. The summon hotkey makes it as quick to reach as a notepad.

One project per part of life

A project for work, one for home, one for a hobby. Each one already knows its background.

Drag, don't paste

When a file is involved, drop it straight in. Faster and tidier than copying text out.

Let the boring bits repeat

Anything you do at the same time each week is a candidate for a scheduled task.

Keep replying

The first answer is a starting point. Say "shorter," "warmer," or "add an example" to steer it.

Start small

Pick one habit this week. Add another when the first feels natural. No need to rush.

The Power Features

Four Tools Worth Learning

Once chatting feels natural, these four turn Claude from a clever helper into something that does real work for you. We will take them one at a time.

01 · Skills 02 · Connectors 03 · Plugins 04 · Routines

Feature 01

Skills:
Claude Makes Real Files

A Skill is an add-on that makes Claude an expert at a specific job. The most useful ones let Claude build proper files for you, not write text about what you should do yourself.

PowerPoint

Slide decks

Excel

Spreadsheets with formulas

Word

Formatted documents

PDF

Reports and forms

Ask "make me a PowerPoint about our Q3 results" and Claude hands back a real .pptx file to download.

Turning Skills On

Three Small Steps

1

Open Settings

Go to Settings, then Capabilities, and switch on "Code execution and file creation." Skills need this to work.

2

Open the Skills list

Go to Customize, then Skills, and toggle on the ones you want, like PowerPoint or Excel.

3

Ask in plain words

Describe what you want. Claude picks the right skill on its own. No buttons to press mid-task.

Skills work on Free and paid plans. Feeling adventurous later? You can even create your own and upload it as a zip file.

Feature 02

Connectors:
Plug In Your Apps

A Connector gives Claude secure access to an app you already use, like Google Drive or your calendar. Instead of copy-pasting, Claude can look things up for you.

Google Drive

Find and read your docs

Calendar

Summarise your week

Gmail

Search and draft replies

Notion, Slack

Read and post updates

Other popular ones include Canva, GitHub, Linear, and Asana.

Adding One

How to Connect

1

Go to Customize, then Connectors.

2

Browse the list and click Connect on the app you want.

3

Sign in to that app and approve access. That is the standard, safe sign-in step.

One thing to know about safety

Claude only gets the same access you already have. If you cannot open a file in that app, neither can Claude.

Sensible habit: only connect apps you trust and actually need. You can switch any connector off at any time, or for one chat only.

Feature 03

Plugins: A Toolkit in One Click

A Plugin is a ready-made bundle of extras you install in one go, instead of setting each piece up by hand. Think of it as a starter kit someone else already packed for you.

Shortcuts

Saved commands for tasks you repeat, triggered with a slash.

Helper agents and skills

Extra specialists and abilities, bundled together and ready to use.

Connections

App connections, all wired up inside the same package.

Worth knowing: plugins live in the more advanced tools, Claude Code and Cowork, not the basic chat box. As a beginner you do not need them yet. In the everyday app, Connectors and Projects already cover most of the same ground.

Feature 04

Routines:
Work That Repeats Itself

A Routine is a task you set up once and Claude runs on a schedule. You write the instructions a single time, then pick how often: every morning, every Monday, and so on.

A daily news brief on the topics you care about

A Monday summary of last week's numbers

A morning rundown of your calendar and inbox

In the desktop app this is called Scheduled tasks (inside Cowork). "Routines" is the matching feature for the developer tool, Claude Code.

Setting One Up

A Scheduled Task in the Desktop App

1

Open Cowork

In the desktop app, switch to the Cowork tab.

2

New task

Click Scheduled in the sidebar, then New task.

3

Write it once

Name it, type the prompt, and pick how often it runs.

4

Save

Claude runs it on schedule and delivers the result.

One catch: a desktop scheduled task only runs while your computer is awake and the app is open. It is a paid-plan feature.

Use It Sensibly

A Few Good Habits

Check important facts

Claude is usually right, but not always. For anything that matters, like health, money, or legal points, confirm it elsewhere.

Mind what you share

Avoid pasting passwords or other people's private details. Treat it like a smart but unknown helper.

Connect only what you trust

Each connector is a standing key to one of your accounts. Add only the ones you genuinely use, and remove the rest.
